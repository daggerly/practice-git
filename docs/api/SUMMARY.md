## 公共错误码
| Code | Message | 语义 |
| :---: | :----: | :--- |
| 0    | Success       | 成功    |
| 1    | Params Error | 参数错误    |
| 2    | Method Error | HTTP方法错误    |
| 3    | Not Found       | 没有查询到数据 |
| 5    | Permission Denied | 权限不够  |
| 99    | Failed| 操作失败  |
| 302    | Location | 重定向，通常由于登录超时  |
